# maze_solver
ROS package for Robotics competition week IITJ

### Instructions
- Clone the repo
```
cd catkin_ws/src

git clone https://github.com/RahulGopathi/maze_solver.git
```
- Do catkin_make
```
cd ..

catkin_make
```
- Source catkin_ws
```
source devel/setup.bash
```
- Launch the world
```
roslaunch maze_solver world.launch
```
